# Finance_Insights


Using: https://pypi.org/project/yfinance/
API

To run

Create virtual environment and then run to activate

python3 -m venv .venv
source .venv/bin/activate

install requirements
pip install -r requirements.txt



install ollama
start the ollama server
ollama pull phi3

vertify the server status
brew services list | grep ollama

pull your model
ollama pull phi3

optional, start a chat to test
ollama run phi3
